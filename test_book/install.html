安装

GitBook安装及使用

安装环境： win10 nodejs(v8.9.3)

  1. 安装gitbook环境
// 执行命令安装cli
> npm install gitbook-cli -g
  1. 初始化项目
// 创建项目目录
> mkdir ./book
> cd book
> gitbook init // 初始化gitbook项目
  1. 构建book项目内容结构

初始化完成后，book项目目录下只有README.md和SUMMARY.md文件 README.md 项目介绍 SUMMARY.md 项目结构定义

> cat SUMMARY.md
# Summary

* [GitBook介绍](README.md)

修改SUMMARY.md

> cat SUMMARY.md
* [GitBook介绍](README.md)
* [安装](install.md)
* [Chapter1](chapter1/README.md)
    * [Section1.1](chapter1/section1.1.md)
    * [Section1.2](chapter1/section1.2.md)
* [发布到GitPage](releasebook.md)

然后重新执行初始化，gitbook会根据SUMMARY.md中的内容生成项目结构

  1. 编译并开启服务
> gitbook build // 编译完成后，会在项目下生成_book目录，
> gitbook serve // localhost:4000
